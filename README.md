# 叠墅空间沟通 MVP

一个用于叠墅装修沟通的 Web MVP。当前阶段聚焦桌面编辑布局、手机查看布局、mock 楼层/家具数据、2D/3D 视图切换占位，以及点击家具查看详情。

## 已实现

- 桌面端三栏布局：左侧楼层栏、中间画布、右侧属性面板
- 手机端布局：顶部楼层选择、中间画布、底部详情抽屉
- 多楼层 mock 数据：B2、B1、1F、2F、院子
- 家具、墙体、房间以结构化 JSON/TypeScript 数据表达
- 2D 底图参考占位，叠加可点击家具、墙体、房间对象
- 3D 白模占位视图，复用同一份结构化数据
- 点击家具后展示唯一编号、名称、楼层、尺寸、材质、备注

## 后续阶段建议

## 运行命令

```bash
pnpm install

```

然后在浏览器打开 `http://localhost:3000`。
