# 林屿湖畔叠墅空间工作台

一个面向自住房装修决策的可视化空间规划工具。项目把四层叠墅的结构、庭院、墙体、门窗、楼梯、家具和后续硬装模块放到同一套结构化模型里，用网页方式完成方案沟通、尺寸推敲、施工表达和分享预览。

在线预览：

https://lyx0599.github.io/lyhpvilla/

## 项目目标

这个工具不是普通效果图展示页，而是一个“业主可持续编辑的装修底盘”。它希望在真实复尺前先完成空间关系推演，在复尺后继续承载硬装模块、点位、清单和施工交底。

当前重点：

- 先建立 B2、B1、1F、2F 和庭院的结构底图。
- 用可点击、可拖动、可校验的对象替代画死在图片里的标注。
- 支持后续把岛台、灶台、马桶、浴缸、淋浴间、衣柜、餐边柜等硬装模块参数化。
- 让方案能被业主、设计师和施工队用同一个链接直观看懂。

## 已实现

- 四层结构工作台：B2、B1、1F、2F 和院子视图。
- 2D 总平面和结构图：墙体、房间、门窗、楼梯、庭院、篱笆、户外铺装分层显示。
- 1F 中间户庭院围合：北院按 2m 进深、南院按 4m 进深，使用木篱笆表达分户和外侧边界。
- 结构绘制工具：直墙、弧墙、隔断、楼梯、篱笆、院子、硬地、小路、绿化。
- 墙线吸附：画墙时可以吸附到墙端点，也可以吸附到现有墙身，方便补墙和连接结构。
- 标签优化：顶部墙体选中后标签自动向下显示，避免被画布裁切。
- 家具和设备占位：家具对象可点击、拖动、旋转，并显示编号、尺寸、材质和备注。
- 多图纸表达：总平面、结构、联动规则、施工标注、家具布置、插座、开关、灯光、水路、排水、吊顶、铺装、效果预览。
- 本地持久化：编辑结果自动保存在当前浏览器，刷新后继续使用。
- 静态发布：Next.js 导出静态站点，并通过 GitHub Actions 发布到 GitHub Pages。

## 设计方向

下一阶段会把“家具占位”升级为“参数化硬装模块库”：

- 厨房：岛台、灶台、水槽、冰箱、高柜、洗碗机、蒸烤箱、操作台。
- 卫浴：马桶、浴缸、淋浴间、台盆柜、壁龛、地漏。
- 收纳：玄关柜、衣柜、餐边柜、家政柜、储藏柜。
- 设备：新风、空调、地暖、净水、弱电、影音设备。
- 庭院：木篱笆、硬地平台、绿化区、户外水电、庭院照明。

模块会逐步支持自然语言创建和修改，例如：

```text
在 1F 厨房门口新增一个尺寸是 2400 x 950 x 900 mm 的岛台
把主卧衣柜深度改成 600
在 2F 主卫加一个 1700 浴缸
```

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Three.js / React Three Fiber 预留
- 静态导出部署

## 本地运行

```bash
pnpm install
pnpm dev
```

然后打开：

```text
http://localhost:3000
```

## 构建发布

```bash
pnpm build
```

构建后会生成 `out` 目录。正式发布使用 GitHub Actions，配置在 `.github/workflows/pages.yml`。推送到 `main` 后会自动发布到：

```text
https://lyx0599.github.io/lyhpvilla/
```

## 数据说明

当前编辑数据保存在访问者自己的浏览器里。分享链接后，每个访问者会拥有自己的本地编辑版本；如果未来需要多人协作同一份方案，需要再接入账号系统和云端数据库。
